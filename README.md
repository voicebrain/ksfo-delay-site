# KSFO Taxi Delays (Public Site)

This is a small static React site that displays the content of:
- `Daily_Delay_Summary` as a searchable/filterable table
- `Statistics` as KPI cards

## Run locally
```bash
npm install
npm run dev
```

## Build for static hosting
```bash
npm run build
```
Deploy the `dist/` folder to any static host (Vercel/Netlify/GitHub Pages/S3, etc).
