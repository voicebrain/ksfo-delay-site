# KSFO Taxi Delays Static Site

This is a **pure static** site (no build step). It loads the included Excel file in the browser and renders the `Daily_Delay_Summary` sheet.

## Files
- `index.html`
- `KSFO_Taxi_Delays_Summary_2026-01-25-shortened.xlsx`

## Deploy on Vercel
When importing to Vercel, set:
- Framework Preset: **Other**
- Build Command: *(leave empty)*
- Output Directory: `.` (dot)
